To add image, we use the [img] tag and it doesn't have a closing tag. The source or path [src] to the images is provided through URL which could be absolute or relative inside [src] attribute. Also, the [alt] attribute is used to provide a description of the image, so that if browser failed to load the image correctly, the alt text would appear and it is also useful for screen readers. [width] and [height] attribute is also often added to specify the image width and height, it is prefarrable to use CSS style to add this, so that internal or external CSS styling will not affect the dimension specified.
