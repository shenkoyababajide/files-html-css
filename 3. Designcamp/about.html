AboutDesign Camp

Design Camp is a coding boot camp organized by experienced tutors, it holds on Obafemi Awolowo University Campus during the usually long end-of-session break.

Interested students are motivated and trained at beginner levels on how to start their coding journey till they become independent to continue to learn and improve on their own.

This website is designed by Shenkoya Babajide Oluwasegun Joel of SBOJ Designs. This website demonstrate the use of purely HTML and CSS to design a static website without any backend functionality.
