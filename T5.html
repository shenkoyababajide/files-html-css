HTML Images

To add image, we use the [img] tag and it doesn't have a closing tag. The source or path [src] to the images is provided through URL which could be absolute or relative inside [src] attribute. Also, the [alt] attribute is used to provide a description of the image, so that if browser failed to load the image correctly, the alt text would appear and it is also useful for screen readers. [width] and [height] attribute is also often added to specify the image width and height, it is prefarrable to use CSS style to add this, so that internal or external CSS styling will not affect the dimension specified. HTML Image Maps

Image maps are used to create clickable areas within an image. The HTML [map] tag defines an image map, an image map is an image with clickable areas. The areas are defined with one or more [area] tags. Generating the coordinates may seem kind of cumbersome but you can generate these coordinated from here

Use of [picture] element for inserting picture

The HTML [picture] element gives web developers more flexibility in specifying image resources. The [picture] element contains one or more [source] elements, each referring to different images through the srcset attribute. This way the browser can choose the image that best fits the current view or choose the next available image should any of the image fails to load.
