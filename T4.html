HTML CSS Styling

CSS is Cascading Styles Sheet. It is used to style html elements to various degrees. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

HTML LINKS

Links are hyperlinks and are found all around every web page. It helps to link one part of the webpage to another in-page or out-of-page source. A link can be a text, an image, a button, smiley and so on... The link tag is the [a], and its most important attribute the the "href="xxx".

By default, links will appear as follows in all browsers:

  1. An unvisited link is underlined and blue
  2. A visited link is underlined and purple
  3. An active link is underlined and red

TEXT LINKS WITH [target] ATTRIBUTE AND ABSOLUTE URL

Values of [href] attribute is a URL which is considered absolute if it links the page to a document on a different website outside. It has the full description path of the site like "https://www.google.com".The [target] attribute when added to the [a] tag helps to determine where the link opens; on the same page, or on a blank page. The default setting is to open the linked document on the same page. [_self] opens the link on the same page, [_blank] opens the linked document on a blank page., [_parent] opens the link on the parent frame, [_top] opens the link in the full body of the window.

TEXT LINKS WITH [target] ATTRIBUTE AND RELATIVE URL

Values of [href] attribute is a URL which is considered relative if it links the page to a document on the same website. It does not have the "https://" part, for example "T2.html".The [target] attribute when added to the [a] tag helps to determine where the link opens; on the same page, or on a blank page. The default setting is to open the linked document on the same page.
